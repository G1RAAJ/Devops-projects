Jenkins Built in node memory configuration
--------------------------------------------
Free Swap Space 2GB
вҡҷпёҸ PART 1 вҖ” Set 2GB Swap
1пёҸвғЈ Remove existing swap (if any)
sudo swapoff -a
sudo rm -f /swapfile

2пёҸвғЈ Create new 2GB swap
sudo fallocate -l 2G /swapfile

3пёҸвғЈ Set permissions
sudo chmod 600 /swapfile

4пёҸвғЈ Create swap area
sudo mkswap /swapfile

5пёҸвғЈ Enable swap
sudo swapon /swapfile

6пёҸвғЈ Verify
free -h

7пёҸвғЈ Make it permanent
echo '/swapfile swap swap defaults 0 0' | sudo tee -a /etc/fstab

вҡҷпёҸ PART 2 вҖ” Set /tmp to 2GB
1пёҸвғЈ Check current mount
mount | grep /tmp

2пёҸвғЈ Edit fstab
sudo nano /etc/fstab

рҹ‘үAdd this line:
tmpfs /tmp tmpfs defaults,size=2G 0 0

to save it, press Ctrl+x, y and m

3пёҸвғЈ Apply changes
sudo mount -o remount /tmp

4пёҸвғЈ Verify
df -h /tmp

set each 1GB as required on jenkins/nodes/Built-in-node/configure
